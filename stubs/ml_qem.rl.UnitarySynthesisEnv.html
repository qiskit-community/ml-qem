ml_qem.rl.UnitarySynthesisEnv¶

class UnitarySynthesisEnv(env_config)[source]¶

UnitarySynthesisEnv.

Example

>>> instruction_set = [
>>>     XGate(),
>>>     CXGate(),
>>>     HGate(),
>>>     U3Gate(Parameter("ϴ"), Parameter("φ"), Parameter("λ")),
>>> ]
>>> render_mode = "console"
>>> env_config = {
>>>     "operator": Operator(random_circuit(2, 3)),
>>>     "n_qubits": 2,
>>>     "model": None,
>>>     "instruction_set": instruction_set,
>>>     "n_instructions": 20,
>>>     "render_mode": render_mode
>>> }
>>> config = (
>>>     PPOConfig()
>>>     .environment(UnitarySynthesisEnv, env_config=env_config)
>>>     .framework("torch")
>>> )
>>>
>>> result = config.build().train()
